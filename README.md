## Overview

A Node.js API client for [Wercker](http://www.wercker.com/)

## Installation

```bash
npm install wercker-client --save
```

## Usage

```javascript
var Wercker = require('wercker-client');
var wercker = new Wercker({
  userName: 'application_owner_name',
  token: 'your_token'
});
```
